Cool Tips and Tricks

Tip 1: Freeze Panes

Freezing panes allows you to keep certain rows or columns visible while you scroll through the rest of the worksheet.

Example:

To freeze the top row, go to View > Freeze Panes > Freeze Top Row. To freeze the first column, go to View > Freeze Panes > Freeze First Column.

Tip 2: VLOOKUP

The VLOOKUP function allows you to search for a value in the first column of a table and return a value in the same row from another column.

Example:

Formula: =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

If you have a table with employee IDs in the first column and names in the second, you can find the name of the employee with ID 123 using: =VLOOKUP(123, A2:B10, 2, FALSE)

Tip 3: Conditional Formatting

Conditional formatting allows you to automatically apply formatting such as colors, icons, and data bars to one or more cells based on the cell value.

Example:

To highlight cells with values greater than 100, select the range and go to Home > Conditional Formatting > Highlight Cells Rules > Greater Than, then enter "100" and choose a format.

Tip 4: Pivot Tables

PivotTables allow you to quickly summarize large amounts of data. You can reorganize and summarize data without altering the original data set.

Example:

Select your data range, then go to Insert > PivotTable. In the Create PivotTable dialog box, choose where you want the PivotTable report to be placed. Drag fields to the Rows, Columns, and Values areas to build your PivotTable.
